# 06 — Naming

> Rev 2 (2026-07-23) — most of the slate is now **settled** (marked ✓).
> Remaining proposals marked ◇ await a9's ack. Rationale unchanged: the
> prototype's names encoded the shallow critique; the real subject is
> existential. Political and geopolitical names largely survive (they
> serve the satire register, which stays).

## Principles

1. **Existential register, not critique register.** Names sound like what
   the entities believe about themselves, not like the author's editorial.
2. **Bland-real corporate texture for companies and products; mythic
   weight reserved for model lines.** Internal codenames may be
   on-the-nose — real ones are.
3. **No real-company puns** (the Palanthropic problem).
4. **Versionable model names** — "X 4.5" must read naturally in a
   MarketWire headline.

## The navigation conceit

The game is *Shoals* — the hazard on the chart. Everyone names themselves
after ways of not running aground, and nobody agrees on where the rocks
are:

- **Shoals** ✓ — the game. The hazard itself.
- **Meridian Capital** ✓ — the fund. The line you navigate by.
- **Polaris** ✓ — the safety lab. The fixed star you steer by: distant,
  dim in bad weather.
- **Halcyon** ✓ — the frontier lab. The one *non*-navigation name in the
  set, deliberately: the kingfisher that nests on the open sea, which
  grows calm *because it is nesting*. The others try to steer past the
  rocks; Halcyon believes that while it is building, there will be no
  storm. Its own hubris, stated as a brand.

## Settled slate

**Halcyon** ✓ — frontier lab. Products as straight as the real world
names things: **Halcyon Chat** (consumer), **Halcyon Teammate**
(enterprise agents — chillingly ordinary on purpose), **Halcyon Compute**
(platform). The recursion product is **Project Fixedpoint** ✓ — internal
codename register, mathematically on-the-nose exactly the way real
codenames are: the model that produces the model, x = f(x). "Fixedpoint"
becomes a household word when it leaks.

**Aleph** ✓ — Halcyon's model line. Borges's point that contains all
points; the transfinite ladder (ℵ₀, ℵ₁, …) is quietly the whole subject;
versions cleanly ("Aleph 4.5"). History ports from Silmarillion's: 1.x
research, 2.x first commercial, current line in production at game start.

**Tianxia** ✓ — Zhaowei Technologies' frontier program ("all under
heaven"); the register was already right. Now the open-weights player
(see [01-world.md](01-world.md)). Model line ✓: **Cangjie** —
the four-eyed legendary inventor of Chinese characters. A language model
named for the man who made language writable; real-world-plausible (the
Cangjie input method exists); open-weights artifacts read right:
"Cangjie-3-235B". (Alts: Luban — patron of craftsmen, open-tools
resonance; Nüwa — she patched the sky.)

**Polaris** ✓ — the safety splinter, per a9 (lab takes the star name).
Model line ✓: **Holt** — a9's suggestion, and it's better than it first
looks: a holt is a small sheltered wood, and an otter's den —
the warm, defensible, deliberately *small* place. The racer names its
model after the totality of all infinities; the safety lab names theirs
after a den. That contrast is thesis-bearing, and "Holt 2" reads
perfectly in a headline. (Alt considered: Walden — right register, but
"Walden 2" collides with Skinner's utopian novel, a resonance we don't
control.)

**Cambria Semiconductors** ✓ — the chip chokepoint (a9's pick; keeps the
Cambrian-explosion shadow at plausible-corporate arm's length). CEO
Helena Karras survives the pass unchanged.

**VXHCN** ✓ — single-name vol index on Halcyon common (HCN), the way real
single-stock vol is watched for the giants. Resolves the rev-1 OPEN: the
index tracks the stock, not a complex — which also proposes that the
game's one chart underlying *is* HCN (see
[03-market-mechanics.md](03-market-mechanics.md)).

**Consensus Markets** ✓ — the prediction-market venue for milestone
binaries. Bland-real. Absolutely not "Delphi."

**Geography** ✓ — Meridia is cut; **Phoenicia** reserved if a Levantine
polity is ever needed. Foreign policy outside Columbia–Serica demotes to
one-liners (decided 2026-07-23; see [01-world.md](01-world.md)). Serica,
Khasuria, Farsistan, Boliviara keep their names for whatever texture
survives.

**Dead names** — Palanthropic/PNTH, Silmarillion, Atlas
Sentinel/Aegis/Companion/Crucible, Covenant, Aletheia, Ptonos, Hearth
(the separate open-weights lab died when open-weights became Tianxia's
identity), Pharos, Lodestar, VXPNT (ticker; the machinery lives on as
VXHCN).
